# Medical Inventory Management System

A C++ Object-Oriented Programming (OOP) project that manages medicine stock, expiry tracking, billing, and reports in real time.

## Features
- Add / Update medicines
- Remove expired items
- Generate low-stock and expiry reports
- Purchase medicines and generate bills
- Log all actions in a history file

## Technologies
- **Language:** C++
- **Frontend:** Console / GUI 
- **Storage:** Text files (`inventory.txt`, `history.txt`)

## How to Run
1. Open `main.cpp` in any IDE (Code::Blocks, VS Code, Dev C++).
2. Compile and run the program.
3. Follow the on-screen menu to manage medicines.
